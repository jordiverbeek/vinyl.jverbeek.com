Voor het starten van de applicatie moet je in de client folder 'npm i' runnen, na het installeren van de node_modules, kan je 'npm run dev' runnen voor het start van de front-end.
Bij het starten van de server, moet je ook eerst 'npm i' runnen, na het installeren kan je 'pm2 start server.js --name api --watch' runnen.

Navigeer naar het aangegeven adres van de front-end, en dan zou de applicatie moeten werken.
